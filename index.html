Installation Guide

How to install and export to Figma

  1. Download the extension files from the button above and extract the archive.
  2. Open the extensions page in Opera or Chrome and enable Developer Mode.
  3. Click Load unpacked and select the extracted ColorPipette folder.
  4. Open any website and launch the ColorPipette popup to scan the page palette.
  5. Click Export to Figma to copy the generated color payload.
  6. Extract the Figma plugin archive and in Figma choose Plugins > Development > Import plugin from manifest.
  7. Select the plugin manifest.json, open ColorPipette Import, and paste the copied code there.
